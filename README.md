# ParametriX
ParametriX is a web content for playing with onomatopoeia. You can experience a variety of visuals and sounds with different parameters.
<br>

# Compatibility 
- Windows10: GoogleChorm, Firefox
- MacOS: GoogleChorm, Safari
<br>
